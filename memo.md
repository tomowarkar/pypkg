## 汎用的な Python プロジェクトのテンプレートを探る

### pipenv の導入

- [Homepage](https://github.com/pypa/pipenv)

```bash
pipenv install --python 3.6
```

- [Homepage](https://github.com/direnv/direnv)

```bash
echo layout pipenv >> .envrc
```
